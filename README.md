# Etch-A-Sketch

A web-based Etch-A-Sketch application that allows users to draw on a grid with customizable features like grid size, color selection, rainbow mode, and eraser functionality. The project is built using HTML, CSS, and JavaScript.

## Features

- **Dynamic Grid**: Adjust the grid size dynamically using a slider (8x8 to 96x96).
- **Color Picker**: Choose a custom color for drawing.
- **Rainbow Mode**: Draw with a rainbow gradient effect.
- **Eraser**: Erase specific parts of the drawing.
- **Reset**: Clear the entire grid.
- **Responsive Design**: Works on both desktop and mobile devices.

## Usage

- Use the slider to adjust the grid size.
- Click the Color button to select a drawing color.
- Enable Rainbow Mode to draw with a rainbow gradient.
- Use the Eraser to remove parts of your drawing.
- Click Reset to clear the grid.
